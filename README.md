# node-ts-start

提供一个 node + typescript 脚手架

## 使用说明

直接 clone 该项目，进行属于你自己的开发。

```bash
git clone https://github.com/yiqingfeng/node-ts-start

npm install

npm run dev
```

## 支持的特性

- node + ts + eslint 开发
- vscode 快启动开发
- utils 通用方法支持（相关API接口文档待完善）
